# Agar.IO Clone

A simple clone of Agar.IO, a popular online multiplayer game where players control cells on a grid and compete to absorb other cells and grow in size.

## Features

- Multiplayer real-time gameplay
- Absorb other players to grow in size
- Simple and intuitive gameplay

## Technologies

- JavaScript
- Socket.IO
- Node.js
- Express.js
- Bootstrap

## Installation

Clone the repository and run `npm install` to install all dependencies.

## Running the Game

Run `npm start` to start the game server. Open two or more web browsers and navigate to `http://localhost:9000` to play the game.

## Development

The game is written in JavaScript and uses the Socket.IO library for real-time communication between the server and clients. The game is divided into three main parts: the client (public folder), the server (servers.js), and the shared game logic (socketStuff folder).
